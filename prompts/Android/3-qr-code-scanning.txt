I would like to add in the feature to be able to import a database config using a QR code, and to share a database config via a QR Code.  

On Android in the functions PhoneDatabaseListLayout and TabletDatabaseListLayout the buttons exist for "Import QR Code". This saves the developer from typing.  

Sharing the config via QR code works the opposite way.  We would take the database config and turn it into a QR code that is then displayed on the screen.
